# Drug-Info-Tool
This is a tool developed for the collection of all information related to a drug like CAS number, Medline Plus ID, Molar mass, InChI key, PubChem ID, Drug Bank ID, ChemSpider, ChEBI, ChEMBL etc. In this tool, a user can type a name a drug or can give a text file input containing a list of the drug.
